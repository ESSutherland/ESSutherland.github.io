In Class Lab Quiz #1

Instructions:

Create a global variable called inputFirstName.

Create a global variable called inputLastName.

Create a global variable called formattedName.

Create a function called loadName( ). The function will do the following:

  • Use a prompt to ask the user to enter a first name.
  • Store the value in the inputFirstName variable.
  • Use a prompt to ask the user to enter a last name.
  • Store the value in the inputLastName variable.

Create a function called formatName( ). The function will do the following:

  • It will accept two input parameters
  • The first parameter should be treated as a first name.
  • The second parameter should be treated as a last name.
  • The function will format the values into last name, first name. Example Smith, Amy
  • Store the result in the variable formattedName.

Create a function called displayFormattedName( ). The function will do the following:

  • It will accept one input parameter
  • The function will display the value of the parameter in an alert with an appropriate message.

User your characterCount( ) so the following button will display the number of characters in the formattedName variable.

Use your uppercaseName( ) so the following button will display the formattedName variable in all uppercase letters.
